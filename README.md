Try On AI - Virtual Fashion Try-On Platform

![Try On AI Banner](https://github.com/user-attachments/assets/4e40c552-2f75-4dd9-8fa5-2e6519eac76a)



> AI-powered virtual try-on platform that lets you visualize outfits, get personalized styling advice, and shop directly from Indian e-commerce sites.


✨ Features

 Virtual Try-On
- Upload portrait + garment images
- AI-generated photorealistic try-ons
- High-fidelity rendering using Gemini AI

 Smart Shopping
- Automatic product detection
- Direct links to Indian e-commerce platforms
- Price comparison and availability

 AI Stylist Assistant
- Context-aware fashion advice
- Personalized recommendations
- Style tips and outfit suggestions

 Personal Gallery
- Save and revisit previous try-ons
- Local storage for privacy
- Export and share options

 Tech Stack

| Layer | Technology |
|-------|------------|
| **Frontend** | React 19, TypeScript, Vite, Tailwind CSS |
| **UI Components** | Lucide React Icons, Custom Components |
| **AI/ML** | Google Gemini API (Vision) |
| **Deployment** | Vercel, GitHub Actions (CI/CD) |
| **Styling** | Tailwind CSS, Custom Animations |

 Prerequisites

- Node.js 18+ 
- npm or yarn
- Google Gemini API Key
- Git
